# Get Started

## Import and Register
```js
import {Button} from 'snowflake-ui'
import Vue from 'vue'

new Vue({
  el: '#app',
  components: {
    's-button': Button
  }
})
```

## Usage
```vue
<template>
  <div>
    <s-button>Default Button</s-button>
  </div>
</template>
```
